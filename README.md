## Recent (public) Work



### Research projects

* [Moving Mahalanobis](https://github.com/QMoayedpour/QMoayedpour/blob/main/files/report_internship_2024.pdf) (Report in French): A method to perform Anomaly Detection on unimodel time series by leveraging a pre-trained model. Online setup enabling continuous inference without any model training.

* [Clinical Trial Duration](https://github.com/QMoayedpour/ClinicalDuration): An econometric project investigating factors prolonging clinical trials. Leveraged NLP techniques and multiple variables to extract features from clinical trial descriptions.

* [Modern Bert for Clinical Trial](https://github.com/QMoayedpour/ModernBert_ENSAE): An application and analysis of ModernBert for clinical trial named entity recognition

* [Online Stochastic Matching](https://github.com/QMoayedpour/OnlineMatchingStochastic): Report and comparison of different methods for online stochastic matching

* [LSTM For Portfolio Allocation](https://github.com/QMoayedpour/Advanced_ML_Ensae): Use of Recurrent-NN Architecture to handle constrained portfolio allocation.

### Implemented Paper and report

* [LongRoPE](https://github.com/Paulcauch/LLM-Project-MVA-2025), Ding & Al. [2024](https://arxiv.org/pdf/2402.13753): A positional encoder to manage high context length (1M+) 

* [Pseudo Inverse Guided Diffusion Models for inverse problems](https://github.com/Juju85194/Generative_Modeling_Project?tab=readme-ov-file), Song & Al. [2023](https://openreview.net/forum?id=9_gsMA8MRKQ): A Problem Agnostic Method to solve inverse problems using diffusion models

* [Detecting changes in slope with an L0 penalty](https://github.com/QMoayedpour/Change-Point-Detection-L1-Penalty?tab=readme-ov-file), Maidstone & Al. [2019](https://arxiv.org/pdf/1701.01672): Changepoint detection with l0 penalty by using dynamic programming

* [Prototypical Networks for Few-shot Learning](https://github.com/NailKhelifa/PrototypicalFewShots?tab=readme-ov-file), Snell & Al. [2017](https://arxiv.org/abs/1703.05175): A neural network designed to handle few shot learning by "learning to separate"

* [Mixture Model for Random Graphs](https://github.com/QMoayedpour/Mixture-Model-for-Random-Graphs), Daudin & Al. [2006](https://inria.hal.science/inria-00070186/document): A statistical model to perform clustering on Unoriented graphs.
